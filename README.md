# 🏆 DK STORE 61

**E-commerce moderno para materiais esportivos desenvolvido por Déryk**

## 📋 Sobre o Projeto

Plataforma completa de e-commerce especializada em:
- 🏈 Camisas de times de futebol e basquete
- 👕 Uniformes de treino (training suits)
- 🛒 Sistema completo de carrinho e checkout
- 📱 Design responsivo e moderno

## 🛠️ Tecnologias

### Backend
- Node.js + Express
- PostgreSQL
- Docker
- JWT Authentication

### Frontend  
- React + JavaScript
- Tailwind CSS
- Context API
- Axios

### Deploy
- VPS própria
- Nginx
- SSL/HTTPS

## 📁 Estrutura do Projeto

```
DK STORE/
├── backend/          # API Node.js
├── frontend/         # Interface React
├── database/         # PostgreSQL + Docker
├── deploy/          # Scripts de deploy
└── docs/            # Documentação
```

## 🚀 Como Executar

```bash
# Em desenvolvimento
git clone [repo]
cd DK STORE
docker-compose up
```
